# terraform-aws-infrastructure
Terraform project to provision AWS infrastructure including VPC, subnets, EC2, and security groups using Infrastructure as Code best practices.

# Terraform AWS Infrastructure

This project demonstrates how to provision AWS infrastructure using Terraform.

## Resources Created
- VPC
- Subnets
- Security Groups
- EC2 instance

## Prerequisites
- AWS CLI configured
- Terraform installed

## Usage

```bash
terraform init
terraform plan
terraform apply
